# Open-Source-Battery-Dataset-Preprocessor

A toolkit for preprocessing open-source battery datasets. 

## Overview
This project aims to convert raw battery data from various formats (e.g., `.csv`, `.pkl`, `.mat`) into a unified, standardized structure. 
The goal is to provide a consistent data foundation for subsequent development of State-of-Health estimation and cycle life prediction algorithms.

## Supported Datasets (Planned)
The project is currently under active development, with plans to support:
